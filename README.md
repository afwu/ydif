# ydiff
diff file1 and file2 after trim/sort/uniq

**download**
```bash
go get github.com/afwu/ydiff
```

**usage**
```bash
$ ydiff -h
what file1 is uniq to file2:            cat file1 | ydiff - file2
what file1 is uniq to file2:            ydiff 1 file1 file2
what file2 is uniq to file1:            ydiff 2 file1 file2
intersect of file1 and file2:           ydiff 3 file1 file2
union of file1 and file2:               ydiff 4 file1 file2

$ echo 123 |ydiff - /etc/hosts
123

$ echo 123 |ydiff 2 - /etc/hosts
# The following lines are desirable for IPv6 capable hosts
# This file was automatically generated by WSL. To stop automatic generation of this file, add the following entry to /etc/wsl.conf:
# [network]
......
```
